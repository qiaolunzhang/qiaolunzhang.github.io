if_nameindex

  • Data Type: struct if_nameindex
    This data type is used to hold the information about a single interface. It has the following members:

unsigned int if_index;
This is the interface index.

char \if_name*
This is the null-terminated index name.

  • Function: struct if_nameindex * if_nameindex (void)
    This function returns an array of if_nameindex structures, one for every interface that is present. The end of the list is indicated by a structure with an interface of 0 and a null name pointer. If an error occurs, this function returns a null pointer.

顺带记一个小问题

前几天安了lantern，每次开机如果不打开lantern，然后关掉，都不能正常上网。后来突然发现atom都不能更新了，提示tunnel失败。刚开始我想可能是lantern使用了隧道。后来发现原来是lantern搞了一个vpn，每次开机都会打开这个vpn，但是lantern如果没有启动，还是用这个vpn上网，就会出现连不了网的情况。
解决办法，搜索network，然后找到vpn选项，关掉即可。

近世代数1——群环域概念

Posted on 2017-11-25 | Visitors:

运算规则

  1. 运算封闭
  2. 结合律
  3. 单位元
  4. 可逆元
  5. 交换律（可逆）
  6. 分配率

群：集合+一种运算（“+”或者”×”）

  • 1 + 2 + 3

  • 结合律
    (ab)c = a(bc)

  • 单位元
    存在一个元素e属于G，是的对任意的a属于G，都有ae = ea = a

  • 可逆性
    对任意的a属于G，都存在a’属于G是的aa’ = a’a = e

环: 集合+两种运算（”+”以及“*”)

  • 对+： 必选1+2+3+4，多了一个交换律
  • 对*: 必选1结合律，可选2或3或4

域： 集合+两种运算（”+”以及“*”)

  • 对”+”： 1 + 2 + 3 + 4
  • 对“*”: 1 + 2 + 3 + 4

称交换环为一个域，如果K中有单位元，且每个非零元都是可逆元，即K对于加法构成一个交换群，K星=K{0}对于乘法构成一个交换群

关于未来的选择

对于未来升学的想法

现在的情况是这样子的，我现在如果想要在交大直升研究生，可能性很小。如果我每门都是95分以上，
这个样子就能达到平均的绩点是3.7左右，但是这样还是很难保研。还是很尴尬。但是如果我申请欧洲的
学校，我成功的概率也不会很小。比方我现在可以找一个实习，一个大公司的实习。

老爸对我的建议

当时我选择信安，其实也是有挺大的原因是因为对它的行业前景看好，但是这个东西真的是我喜欢的吗？
我愿意为之付出终身的研究吗？这个东西我没有考虑过。
但是结果我真的走进了这行，才发现信安也是一个大坑，信安的课程安排很不合理，学习的更多的是通信
的课程，而现在我对信安其实也没有由衷的喜欢。

以前我是觉得擅长理科，擅长数学是件非常值得骄傲的事情。当时觉得信安会有挺多数学课，是个可以
挑战自己的机会，就去了信安。但是我真的会喜欢编程吗？我当时甚至都没有学过编程。从这看其实我
当时想法很幼稚。

同时还有一个原因，是我当时对被调剂到生物很反感，就一直想走。

其实要做出一个选择的时候，将就很可能会导致之后的反悔。譬方今天性与健康课，老师讲，很多男人
在选择妻子的时候，会更多考虑理性的因素，譬方这个女人是不是顾家，却往往忽略了自己内心的想法。
忽略了自己究竟是喜欢什么样的菜。跟着自己的心走，才不会走丢。

就像现在老爸跟我讲，要去东南大学读信安。

我不想去，有两点。

  • 信安我真的想学吗？
  • 我心理上接受不了
    在我看来，这已经构成我不想去的充分必要条件了。

至于以后的选择

尽人事，听天命。
跟着自己的心走, 不要主观的对职业做一个心理上的限定，比方什么是高大上，其实自己喜欢更重要。
